# Operate
Operate for FreeNAS is an unofficial app that let's you perform certain admin functions to your FreeNAS servers.

Features:

Accounts
- Groups
- Users

Storage
 - Pools
 - Snapshots
 - Disks

Services
Plugins
Jails

Alerts
Updates
Reboot
Shutdown
